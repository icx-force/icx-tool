# icx-tool
Tools for ICON

## Requirements

- Python 3.9 or later.

## Install
```shell
$ pip install icx-tool
```

## Usage
```shell
$ icx-tool --help
Usage: icx-tool [OPTIONS] COMMAND [ARGS]...

Options:
  --url TEXT  JSON-RPC URL. Predefined: "local", "mainnet", "berlin", "lisbon"
  --help      Show this message and exit.

Commands:
  ks    Command for key store
  prep  Command for P-Rep
$ 
```
